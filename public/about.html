Tweet Visualizer App

This application uses the Express framework , Bootstrap and d3.js to build a tweet visualization app that is deployed to AWS EC2. The application stores data in Elastic Search

Features

  • Live tweets for few keywords are indexed in elastic search using logstash
  • App provides UI visualization for the tweets. It provides dynamic charts showing top hash tags
  • Data can be explored using charts
